# college-scorecard
Designed and implemented a graduate-level Oracle SQL data-analysis project using U.S. Department of Education’s College Scorecard 2023 dataset (~6 000 institutions).
Applied advanced SQL for institutional performance, access, and equity analysis across states and sectors.

End-to-end Oracle SQL project demonstrating advanced query design and higher-education data analytics.
Includes schema script, analytic queries, and performance-equity dashboards built directly from federal open data.
